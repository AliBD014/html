Headings

HTML provides six levels of headings, from <h1> to <h6>.

<h1>This is an H1 Heading</h1>
<h2>This is an H2 Heading</h2>
...
<h6>This is an H6 Heading</h6>

Paragraphs

Paragraphs are created using the <p> element.

<p>This is a paragraph of text.</p>

Links

Links are created using the <a> element.

<a href="https://example.com">Visit Example</a>

Images

Images are added using the <img> element.

<img src="image.jpg" alt="Description of image">

Videos

Videos are added using the <video> element

<video>
<source src="vid.mp4"> 
</video>

Tables

Tables are created using the <table> element.

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Forms

Forms are created using the <form> element.

<form action="submit.html" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Submit</button>
</form>
